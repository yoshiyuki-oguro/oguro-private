# ② 自分史

これまでやってきたこと、そしてこれから起きることを、人生スパンでまとめる。発表するつもりはない、自分のための記録。**長期の振り返り**。

## 書き方の方針（重要）

年表（箇条書き）ではなく、**時系列を守った「ストーリー（物語）」として綴る**。
②の一次資料「創業者小黒正信の信念を共有する会」で本人が書いたような、出来事とそのときの心境がひと続きに流れる語りの形。

各時期で、次の3つが物語の中に織り込まれるようにする。

1. **役割の変化** — 立場・肩書き・担っていたこと
2. **心境の変化** — そのとき何を感じ、何に悩み、何を目指していたか
3. **外部環境の変化** — 会社・業界・家族・世の中の動き

## 運用ルール

- `自分史.md` を時系列のストーリーとして書き、**都度「追記」していく**（前の記述は基本残す）。
- すっかり忘れてしまった期間は、無理に埋めず「空白」として残してよい。思い出したら足す。
- ①10大ニュースやNotion日記、Evernoteの書き出しが、記憶を呼び起こす手がかりになる。
- Claudeは事実の羅列でなく、「そのときの心境」を引き出す問いを添えて伴走する。
- Evernote等のHTML書き出しは `scripts/html_to_text.py` でテキスト化してから読み解く。

## 一次資料（読み解き済み・ストーリー化の素材）

- `20260612 bondex α 冒頭小黒の話/` — 社長就任(2015)〜bondex(2026)の歩みを「リーダー＝覚悟」で貫いた年表的な語り。
- `創業者小黒正信の信念を共有する会/` — 2021年の「覚悟が定まった」内面の転機（若葉の峠／なんで私が東大に!?／創業者「釘を打ちたくない」の真意への気づき）。
