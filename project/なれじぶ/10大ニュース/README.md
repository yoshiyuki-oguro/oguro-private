# ① 10大ニュース（10大日記）

その年に自分に起きたニュースを、起きたときに書き留めていく。年明けに1年を振り返り、10個を選んで順位をつける。**短期（1年サイクル）の振り返り**。

## 入力の2パターン

出来事を `YYYY_ニュースメモ.md` に貯めていく方法は2つ。両方を併用する。

- **(A) 都度報告**：小黒さんが「これは今年のニュース」と感じた出来事を伝える → Claudeが事実＋心境で追記する。
- **(B) 定期レビュー提案**：Claudeが定期的にNotion日記を読み、ニュース候補を拾って小黒さんに提案する → 小黒さんが採否を判断し、採用分を追記する。心境が日記から読み取れれば併せて下書きする。

## 運用ルール

- 年の途中：`YYYY_ニュースメモ.md` に、起きた出来事を随時 append で書き留める（順位はつけない）。
- 年明け：メモを見返し、`YYYY_10大ニュース.md` に10個を選んで順位を確定する。
- 出来事は「事実」だけでなく「そのときの心境」も一言添えると、後で②自分史に活きる。
- Notionへの書き込みは承認制（読み取りは可）。
