# 社史 ― ボンド商事／bwdグループの歩み

②自分史とは**別軸**の記録。主語は「私（義幸）」ではなく、**会社・創業者**。
事実（年月・人物・出来事・意思決定）を時系列で記録する。

## 自分史との関係

- 一次資料は**両方で共有**する（現物は下記に集約）。
- 社史 → 事実として整理・記録する。
- 自分史 → 社史の中から「義幸の覚悟・価値観に響いた部分」だけを引用し、"私の物語"に織り込む。

## 位置づけ

いまは個人ワークスペース（なれじぶ）内に置く。将来、会社の正式な社史として使うなら、共有リポジトリへ昇格させることも検討する。
