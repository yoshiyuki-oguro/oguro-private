# uncle b株式会社 — コンセプト・戦略まとめ

> 策定：2025年5月　bwdホールディングス株式会社

---

## 1. 背景と課題

ボンド商事株式会社において、50年にわたり接着剤・建材業界に貢献してきた人財が、2025年5月をもって円満退社を迎えた。

この方は業界随一の商品知識・仕入れ先との深い信頼関係・広範な人脈を持ち、会社としてはその経験を社外に流出させたくないという思いが強い。一方、同じ組織内に残すと世代間ギャップが生じやすく、本来の力が発揮されないリスクがある。

そこで、**別法人を活用した柔軟な協力関係の構築**を検討することとなった。

---

## 2. 解決策：uncle b株式会社の活用

bwdHDが100%出資する休眠会社「uncle b株式会社」（資本金300万円）を再稼働させ、退職者との業務委託契約の受け皿とする。

| 項目 | 内容 |
|---|---|
| 会社名 | uncle b株式会社 |
| 資本金 | 300万円 |
| 株主 | bwdホールディングス株式会社（100%） |
| 活用目的 | OBの顧問契約受け皿・新事業運営主体 |

---

## 3. 事業モデル

### Phase 1：顧問業務（アドバイザリー）

退職者と顧問業務委託契約を締結。**非常勤顧問として月額5万円**（消費税別）を支払い、下記業務を委託する。

- 経営・業務上の課題へのアドバイス・提言（主に代表からの依頼）
- 業界動向・市場情報・技術情報の提供
- 取引先・業界関係者との関係維持・橋渡しサポート
- プロジェクト・商品開発・調達に関するアドバイス

インセンティブ報酬は成果に応じて別途書面合意。雇用ではなく業務委託のため、指揮命令関係は生じない。

### Phase 2：中古ゴルフショップ事業

当該人財の深いゴルフギアの造詣を活かし、uncle bが中古ゴルフショップ事業を展開する。

- 当初は**オンライン販売からスタート**し、将来的な実店舗化も検討
- 「**目利きのプロが選び・語り・届ける**」ことで生まれる付加価値が強み

### Phase 3（将来構想）：多様な人財の集積プラットフォーム

uncle bをbwdグループに限らず、様々な業界で経験を積んだ人財が集まるプラットフォームに育てる。一線を退いてもまだ輝ける人財を広く受け入れ、その知識・経験・人脈を社会に還元する場とする。

## 4. フィロソフィー＆ブランドアイデンティティ

### コーポレートフィロソフィー

> **温故知新**
>
> 先人が築いてくれたものを、次の時代につなぐ。  
> 知識、経験、人脈、志——それらは組織の中で眠らせるには惜しいものだ。  
> uncle bはそれらを受け取り、形を変えて社会に還す。  
> 誰かの長い年月が、誰かの明日の力になる。それがuncle bの使命だ。

### 社名の由来

アメリカの国民的シンボル「Uncle Sam」にならい「uncle b」と命名。「b」にはふたつの意味が込められている。

| b | 意味 |
|---|---|
| **b as bond** | 1953年創業以来、接着剤・建材の世界で信頼を積み重ねてきたボンド商事の頭文字。先人が築いた事業と志の遺伝子。 |
| **b as Bridge** | 過去と未来、人と人、知識と社会をつなぐ「橋」。先人の遺産を次世代へと受け渡す架け橋。 |

### ロゴに込めた想い

| 要素 | 意味 |
|---|---|
| **オレンジ** | オールドアメリカン風の温かみある色。Uncle Samのスピリットとボンド商事の歴史を象徴。 |
| **円（丸）** | 裏メッセージとして「日の丸」を表現。日本の誇りある職人・人財への敬意。 |
| **足跡** | 温故知新の体現。先人が歩んだ道の証を踏みしめ、次の一歩を踏み出すイメージ。 |
