### 確定事項（たたき台に反映済み）
| 項目 | 内容 |
|---|---|
| 会の名称（仮） | bwdグループOB会 / bwd Group Alumni Association |
| 正会員 | 退職済みOB・OG（通算5年以上 or 定年退職、同業他社移籍なし、懲戒解雇でない） |
| 準会員 | 定年後も嘱託社員として在籍中の方（通算5年以上、同業他社移籍なし）→退職後は正会員に自動移行 |
| 名誉会員 | 幹事会推薦・会費免除 |
| 年会費 | 正会員・準会員ともに2,000円、名誉会員無料 |
| 会計年度 | 毎年4月1日〜翌年3月31日（設立当初は設立総会日〜翌3月末） |
| 第２条の会社列挙 | 削除済み（「グループ各社」と総称。事業会社は増減する可能性があるため） |

### 残検討事項（たたき台に「※検討事項」として記載）
- 会の名称・略称の正式決定（設立総会で）
- 「同業他社」の具体的定義・対象企業リストの作成
- 同業他社退職後の再入会可否
- 入会金の要否
- 役員の選出方法（互選・推薦・投票）
- 設立総会の開催時期・形式・設立準備委員会の組成
- SNS・連絡ツールの選定
- 海外拠点（B&I Vietnam）在籍者の扱い

### セッション中の気づき（次回への引き継ぎ）
- 「準会員」の定義は最初に確認すること。今回は「現役社員全般」として書いてしまい、後から「嘱託社員のみ」に修正した
- 条文内に「（正会員③）」のような内部参照記号を書くと読んだ人に意味不明になる。平文で「通算5年以上在籍していること」のように書く
